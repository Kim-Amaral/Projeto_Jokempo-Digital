# Projeto_Jokempo-Digital
Implementação digital de um jogo de Jokempo com uso de uma FPGA, uso da linguagem de descrição de hardware VHDL, projetado com auxílio do software Quartus.

O arquivo PDF "Relatório_Projeto" apresenta o relatório descrevendo toda a descrição e design do projeto passo-a-passo, já o arquivo QAR "projeto_exp9" apresenta o design dos circuitos digitais no Quartus.
